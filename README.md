maven depency

Add jitpack repo

```xml
	<repositories>
		<repository>
		    <id>jitpack.io</id>
		    <url>https://jitpack.io</url>
		</repository>
	</repositories>
```

add dependency

```xml
	<dependency>
	    <groupId>com.github.ZooMMaX</groupId>
	    <artifactId>TelegramBotApp</artifactId>
	    <version>29e47b1710</version>
	</dependency>
```
